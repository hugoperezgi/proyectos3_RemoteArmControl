Remote Operation of a Humanoid Robot Arm

The paper of this project can be found here.

Setup

For convenience, a second laptop was used as a power supply for the NodeMCU board. This is because, while the board could be powered using an external power supply, it's easier and faster to use a USB cable to provide power plus, a USB connection is required to upload new code versions to the NodeMCU board.
